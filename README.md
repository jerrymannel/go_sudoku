# go_sudoku
Trying to make a fast sudoku solver using Golang

```
./goSudoku -i 058409020000060089201030700000013400320080901079000608060005817815700304030090200 -o 658479123743162589291538746586913472324687951179254638962345817815726394437891265
```


```
$ ./goSudoku
  -d	Show iterations
  -g	Show grid
  -i string
    	Input puzzle
  -o string
    	Expected output
```

Use `getData.sh` to generate a new set of test data
